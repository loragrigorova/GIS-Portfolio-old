Overview

This page complements the GIS portfolio by presenting a vector-based interactive map. The map visualises spatial features (points, lines, and polygons) and demonstrates common vector workflows such as styling, attribute-driven symbology, and basic interaction. It is intended as a concise demonstration of how vector data can be prepared, published, and explored on the web.

Methods

Vector data were prepared and processed using desktop GIS (for example, QGIS) and exported to GeoJSON for web use. Key steps included cleaning and validating geometries, simplifying complex shapes for web performance, and joining external attribute tables for thematic mapping. The web map can be implemented with libraries such as Leaflet or Mapbox GL JS to render GeoJSON, apply attribute-based styling, and enable popups and tooltips for feature inspection.

Results

The implemented map allows users to pan and zoom across the study area, inspect feature attributes via popups, and view different symbology based on attribute values. Geometry simplification and efficient data formats keep the map responsive at common zoom levels while preserving meaningful spatial detail. The visualisation communicates spatial patterns and attribute differences clearly to viewers.

Reflection

Working with vector data for web mapping highlights trade-offs between visual detail and performance. Simplification, tile-based approaches, or server-side vector tiles may be required for large datasets. Future enhancements could include interactive filters, a legend explaining symbology, data download options, and accessibility improvements to broaden usability.
