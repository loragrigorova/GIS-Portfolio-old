Vegetation Indices Analysis

Compare NDVI (Normalized Difference Vegetation Index) changes from 2017 to 2025 in the Congo region.

About This Analysis

This interactive map displays vegetation indices derived from satellite imagery. NDVI values range from -1 to 1, where higher values indicate denser vegetation. Use the checkboxes to toggle between different imagery layers and the Compare button to activate the side-by-side swipe feature.

Key Features:

  • NDVI (2017 & 2025): Normalized Difference Vegetation Index showing vegetation health and density
  • True Color: Natural color imagery for reference
  • Change Detection: Visualization of vegetation change over 8 years
  • Side-by-Side Compare: Drag the slider to compare two layers directly
